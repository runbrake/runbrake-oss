# RunBrake OSS v0.1.0 Developer Preview

RunBrake OSS v0.1.0 is the first public developer preview of the local scanner, local sidecar, and OpenClaw policy-plugin adapter.

## Install

Install with Homebrew:

```bash
brew install runbrake/tap/runbrake
runbrake
```

Or use a signed or checksummed binary release:

```bash
curl -fsSLO https://github.com/runbrake/runbrake-oss/releases/download/v0.1.0/runbrake_v0.1.0_darwin_arm64.tar.gz
curl -fsSLO https://github.com/runbrake/runbrake-oss/releases/download/v0.1.0/runbrake_v0.1.0_checksums.txt
grep " runbrake_v0.1.0_darwin_arm64.tar.gz$" runbrake_v0.1.0_checksums.txt | shasum -a 256 -c -
tar -xzf runbrake_v0.1.0_darwin_arm64.tar.gz
./runbrake doctor --path ~/.openclaw
```

## First Commands

```bash
runbrake doctor --path ~/.openclaw
runbrake assess --path ~/.openclaw --format markdown --output runbrake-assessment.md
runbrake scan-skill ./skills/my-skill --format sarif --output runbrake.sarif
runbrake sidecar --policy ./examples/policies/block-critical-installs.json
```

## Commercial Split

This release includes local sidecar enforcement and local audit events. It does not include hosted dashboards, Slack/web approval workflows, private catalogs, retained hosted audit history, enterprise integrations, or hosted isolation. Those belong to the commercial RunBrake control plane.
